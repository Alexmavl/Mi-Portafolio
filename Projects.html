Proyectos Realizados

Proyecto No. 1

Este proyecto se trabajó con el lenguaje C#, en grupo, para resolver de la manera más eficiente los siguientes métodos numéricos:
- Método Newton Raphson
- Método de la Secante
- Método Müller
- Método Gauss Jordan

Se incluye el método de inicio de sesión, el cual se creó con una base de datos con los usuarios y contraseñas encriptadas.
Contiene un menú principal, el submenú de los métodos y la información de los integrantes.

Proyecto No. 2

Este proyecto está elaborado con el lenguaje C# utilizando el modelo MVC de .NET Framework, relacionado con un gabinete de abogados.
Consta de dos tablas: Cliente y Asunto, se realizó el CRUD para cada tabla. La base de datos utilizada fue en SQL Server.
